Vấn Đề Của Merchant

  • Shopify không có return portal cho khách hàng — mọi return đều qua email/ticket thủ công
  • Xử lý 1 return request tốn trung bình 15-20 phút nhân lực
  • Phần lớn merchant xử lý return bằng email + spreadsheet — dễ sai sót, mất tracking
  • Các app đối thủ giá $19 - $599/tháng, nhiều app tính thêm phí per-label
  • Không có app nào có exchange flow tốt — đa số chỉ refund rồi tạo đơn mới
  • Merchant mất doanh thu vĩnh viễn khi refund 100% thay vì convert sang store credit hoặc exchange

Cơ Hội Cho Avada

3.9M Shopify stores — trong đó 1.17M store có >50 đơn/tháng cần return management. Phần lớn app đối thủ tập trung Shopify Plus ($99+/tháng). Thị trường SMB ($0-30/tháng) gần như bỏ trống — đây chính là phân khúc Avada mạnh nhất.

Giải Pháp: Avada Returns & Exchanges

Self-service portal + Revenue retention engine

Cho Khách Hàng

  • Self-service return portal — tự tra cứu đơn hàng
  • Chọn item muốn trả, nhập lý do + ảnh chụp
  • 3 lựa chọn: Refund / Store Credit / Exchange

  • Tự in shipping label hoặc nhận QR code

  • Cấu hình return policy linh hoạt (theo product/tag/collection)
  • Auto-approve return theo rules
  • Store credit incentive — thưởng bonus khi chọn credit
  • Tự động tạo shipping label (USPS, FedEx, UPS, DHL)
  • Analytics: return rate, revenue retained, top reasons
  • Email notifications tùy chỉnh template
  • 10-phút quick-start wizard

P1 Competitive Advantage

  • Instant Exchange — Ship mới trước khi nhận return
  • Cross-product Exchange — Đổi sang sản phẩm khác
  • Shop Now Exchange — Khách browse catalog khi exchange
  • Fraud Detection — Chặn serial returner, blocklist
  • QR Code Returns — Box-free, in-store drop-off
  • SMS Notifications — Twilio integration
  • Green Returns — Keep-item (dưới ngưỡng giá)
  • Shopify Flow — Trigger automation

P2 Enterprise & Scale

  • Multi-warehouse Routing — Return đến kho gần nhất
  • Cross-border Returns — Quốc tế, customs handling
  • In-store Returns (POS) — Nhận return tại store
  • Warranty Management — Claims theo product warranty
  • White-label Portal — Custom domain
  • API & Webhooks — Tích hợp ERP/3PL
  • Advanced Analytics — Product-level insights

Revenue Retention Engine

Thay vì refund 100% (mất doanh thu), app khuyến khích khách chọn store credit bằng cách thưởng bonus (VD: hoàn 110% dưới dạng credit). Kết hợp với exchange flow giúp merchant giữ lại trung bình 40% doanh thu từ các đơn return. Đây là USP chính so với đối thủ giá rẻ.
