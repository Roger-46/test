## 1. AfterShip Returns & Exchanges

**URL:** https://apps.shopify.com/returns-center-by-aftership
**Tagline:** Retain traffic & revenue with exchanges and store credits
**Developer:** AfterShip (Singapore, since Feb 24, 2017)
**Screenshot:** [01-listing-page.png](aftership/01-listing-page.png)

### Pricing
| Plan | Price | Highlight |
|------|-------|-----------|
| Returns Core | **Free** | 3 returns/month, branded returns portal |
| Essentials | **$11/month** | 30 returns/month, automated approval, basic features |
| Pro | **$59/month** | 200 returns/month, advanced rules, integrations |
| Premium | **$239/month** | 1,500 returns/month, automated workflows |

### Tech Stack & Integrations
- **Languages:** English, German, French, Spanish, Italian (5 languages)
- **Works with:** Shopify POS, Shopify Flow, Shopify Admin, Canada Post, FedEx, Geodis, UPS, USPS

### Key Features (from listing)
- Reduce return costs and recapture more revenue with automated returns and exchanges
- Self-service returns portal with branded return page
- Automate return approvals, refunds, and updates
- Customer support: identify faulty products and bring losses with built-in analytics
- Deliver seamless return experience with branded return pages and native POS solution
- Identify return patterns and time business strategy with built-in analytics
- Offer free returns and offset the cost with Return Care

### Strengths
- Market leader (most reviews by far - 1,251)
- Built for Shopify badge
- Multi-language support (5 languages from Free plan)
- Backed by AfterShip's mature shipping/tracking infrastructure
- Cheapest entry-level paid plan ($11/mo)

### Weaknesses
- Free plan very limited (only 3 returns/month)
- Best features locked in Pro/Premium ($59-$239)
- Not as deep on exchange flow as Loop

### Key Features (from listing)

## Insights cho Avada Returns & Exchanges

### 1. Gap thị trường mà Avada nên đánh
- **AfterShip** chiếm thị phần SMB nhưng free plan quá ít (3 returns)
- **Loop** quá đắt cho SMB ($155 entry)
- **Return Prime** giá tốt nhưng có complaints về quality

→ **Avada cơ hội:** Free plan **50 returns/tháng** (gấp 10-15x AfterShip), entry plan $9-29 (cạnh tranh trực tiếp Return Prime), tập trung Quality + Vietnamese support tốt.

### 2. Tính năng phải có (table stakes)
Tất cả 3 đều có:
- Self-service portal
- Branded return pages
- Automated return labels (USPS, FedEx, UPS)
- Store credit + bonus
- Exchange flow
- Email notifications
- Analytics dashboard

### 3. Tính năng khác biệt hóa
- **Loop** dẫn đầu: AI fraud detection, Shop Now exchange, order editing
- **AfterShip** dẫn đầu: Multi-language (5 ngôn ngữ trên free plan), POS integration
- **Return Prime** dẫn đầu: 100+ integrations

→ **Avada nên copy:** Multi-language từ AfterShip, Shop Now flow từ Loop, integration depth từ Return Prime.

### 4. Pricing strategy
Để cạnh tranh tốt, Avada nên:
- Free: 50 returns/tháng (đột phá so với 3-5 của đối thủ)
- Starter: $9-19/tháng — đè AfterShip $11 và Return Prime $19.99
- Pro: $49-79/tháng — đè Return Prime $49.99
- Enterprise: $149-199/tháng — rẻ hơn Loop $155 nhưng đủ tính năng

### 5. Marketing angles
- "Free plan gấp 10x AfterShip"
- "Loop features at SMB pricing"
- "Quality alternative to Return Prime"
- Vietnamese-first support
