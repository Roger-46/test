1. Nghiên cứu thị trường & phân tích đối thủ BA chịu trách nhiệm hiểu người bán (merchants) và đối thủ cạnh tranh: Phân tích top apps cùng category trên Shopify App Store. Ghi lại tính năng, pricing, flow, review insights. Tìm gaps để đề xuất tính năng mới. Theo dõi trends về UX/UI của Shopify. Deliverables: Market report Competitor comparison Opportunities & pain-points document 2. Phỏng vấn người dùng & phân tích nhu cầu BA trực tiếp nói chuyện với merchants để hiểu: Họ đang gặp vấn đề gì trong business. Họ mong muốn app giải quyết điều gì. Các workflow của họ (cart → checkout → fulfillment → marketing). Mức độ tech-savvy của từng nhóm merchant. Deliverables: Customer insights report Job-to-be-done statements User persona Customer journey map 3. Viết requirement rõ ràng cho team dev Đây là phần quan trọng nhất của BA: Chuyển nhu cầu user → requirement kỹ thuật rõ ràng. Viết PRD, BRD, Use case, User flow. Mô tả logic + rules + edge cases. Viết API requirements nếu tính năng cần backend. Deliverables: PRD (Product Requirement Document) Wireframe / Flow Diagram (Figma) Acceptance criteria cho mỗi task API contract (nếu cần) 4. Thiết kế flow & UX cơ bản Không phải designer nhưng BA thường: Vẽ user flows, upsell funnels, settings flow. Đề xuất UX hợp lý theo chuẩn Shopify Polaris. Kiểm tra UI có đúng guideline Shopify App Store. Tools thường dùng: Figma, Miro, Flowchart, Whimsical. 5. Viết task cho dev (ticket breakdown) BA là người chuyển PRD → tasks nhỏ cho dev: Backend tasks Frontend tasks API tasks Data tasks QA test cases Deliverables: Jira tickets / Linear tasks Business logic mô tả cực rõ ràng Acceptance criteria (Given – When – Then) 6. Làm việc với dev & designer để đảm bảo đúng requirement BA cần: Giải thích logic cho dev trong sprint. Review UI mà designer làm. Họp refinement: hiểu feasibility, cost, risk. Ưu tiên backlog theo impact/effort. 7. Kiểm thử (UAT) & đảm bảo chất lượng BA là người test app đầu tiên: Test theo use-case thực tế. Test Shopify checkout, cart, discount, API. Test merchflow: product, variants, orders, shop metafields… Test compatibility: theme, cart types, markets… Deliverables: UAT checklist Bug reports Feedback cho dev 8. Viết tài liệu hướng dẫn Bao gồm: Onboarding content Help docs Tutorial videos In-app tooltip text Release notes 9. Tối ưu conversion & retention BA theo dõi: App install → activation funnel Feature usage tracking Drop-off points Merchant behavior Và đề xuất cải tiến: UI optimize Feature enhancements Onboarding flow Pricing optimization 10. Phân tích dữ liệu & quyết định sản phẩm BA xem: Daily Active Stores Activation rate First Upsell Order (FUA) Retention rate Revenue per merchant Feature adoption Deliverables: Monthly product report Churn analysis Feature impact analysis 11. Hỗ trợ merchant & support team BA hỗ trợ team Customer Support khi: Merchants gặp lỗi. Cần giải thích luồng xử lý. Cần debugging behavior. BA giúp: Giảm ticket volume Nâng cao CSAT Giải thích rõ ràng logic sản phẩm 12. Chuẩn bị app cho Shopify App Review BA kiểm tra: UI theo Polaris Privacy policy Navigation Pricing display Data access & webhooks App performance GDPR compliance
