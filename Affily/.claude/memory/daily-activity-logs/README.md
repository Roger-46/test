# Daily Activity Logs - Affily

Folder lưu trữ các file log theo từng ngày làm việc trên dự án Affily.

## 📁 Cấu trúc

```
daily-activity-logs/
├── README.md              ← Hướng dẫn này
├── 2026-03-14.md         ← Log của ngày 14/3
├── 2026-03-15.md         ← Log của ngày 15/3
└── ...
```

## 📝 Format File

**Tên file:** `YYYY-MM-DD.md`
- Ví dụ: `2026-03-14.md`, `2026-03-15.md`

**Nội dung:**
```markdown
# Daily Activity Log - 2026-03-14

## ✅ Completed
- [x] Task 1
- [x] Task 2

## 📋 Active
- Current work

## ❌ Blockers
- Issues

## 🎯 Next Steps
1. Action 1
2. Action 2

## 💡 Learnings & Insights
- What I learned today
- Patterns discovered
- Best practices identified
- Improvements for next time

## 📊 Status Summary
- Overall project status
- Key metrics
- Progress notes
```

## 🚀 Cách sử dụng

1. **Mỗi ngày có hoạt động**, tôi tạo file mới: `YYYY-MM-DD.md`
2. **Ghi nhận:**
   - Completed tasks ✅
   - Active work ⏳
   - Blockers ❌
   - Next steps 🎯
3. **Không có activity:** Không tạo file
4. **Status Summary:** Project health at a glance

## 💾 Archive

Sau 1 tháng, file cũ có thể move vào folder `archive/` để keep folder clean.
