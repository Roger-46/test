**Trigger**: Roger noi "tao task notion va jira"

**Thu tu**:
1. Tao Notion task truoc -> lay URL
2. Tao Jira Sub-task voi description = `Notion: {notion_url}`
3. Bao ca 2 link trong 1 tin nhan

**Title**: Dung cung title cho ca 2, Jira them tien to `[BD][AFF]`
**Noi dung**: Neu Roger khong chi dinh -> Notion de trong, Jira description = link Notion
