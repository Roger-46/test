Feature folders trong `documents/roger/features/` phải đánh số đằng trước: `{số}-{tên-feature}`

Thứ tự theo product flow:
1. dashboard
2. conversion-page
3. auto-tag-referral-orders
4. payment-schedule
5. email-notifications
6. forgot-password
7. research-feature

**Why:** Roger muốn thấy thứ tự rõ ràng trong file explorer.

**How to apply:** Khi tạo feature folder mới, đánh số tiếp theo (8, 9, ...). Thứ tự theo product flow, không theo thời gian tạo.
