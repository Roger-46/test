Khi Roger yêu cầu cập nhật scope/sửa tính năng bình thường → chỉ sửa **PRD + UI mockup**.

**Why:** Các file khác (research, test case, Notion, Jira) sẽ được cập nhật riêng khi cần. Sửa tất cả mỗi lần thay đổi scope nhỏ = tốn thời gian không cần thiết.

**How to apply:**
- Yêu cầu bình thường ("cập nhật lại", "sửa lại") → chỉ PRD + UI mockup
- Yêu cầu rõ ràng ("sửa toàn bộ file", "cập nhật hết") → sửa tất cả files liên quan (research, PRD, UI, test case, Notion, Jira...)
